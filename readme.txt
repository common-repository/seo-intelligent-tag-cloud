== Description ==

SEO Intelligent Tag Cloud is the best plug-in for list of tags in what is called a 'tag cloud'. The tag is added only when they reach a minimum number of posts attributed to him, this number is defined by the user (customizable). This helps your ranking on search engines  because they are not  generated  within  a few  pages  with content tags. Search engines will thank you!

In the options panel for the plugin (widget), you must insert the minimum number of posts attributed to every tags for stamp online.

* Other Server to [Download SEO Intelligent Tag Cloud](http://www.gb-rugs.com/plug-in/seo-intelligent-tag-cloud.zip)
* Other [Wordpress plugins](http://www.prima-posizione.it/download/wp/) by the same author.

== Installation ==

This section describes how to install the plugin and get it working.

1. Delete any existing `seo-intelligent-tag-cloud` folder from the `/wp-content/plugins/` directory
2. Upload `seo-intelligent-tag-cloud` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the widget options panel and drag-and-drop the Widget in your right-sidebar
5. Insert the minimum posts attributed to any tag (min. 8 recommended).
